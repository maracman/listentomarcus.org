# Marcus Anderson Personal Webpage

Static GitHub Pages site for Marcus Anderson research and archives.

- `index.html`: research homepage with links to current working papers.
- `composer-archived/`: archived clone of the former Marcus Anderson music site.
- `music/`: local legacy copy of the archive from the first build pass.
- `assets/music/`: media recovered from Squarespace CMS and archive references.

When the hosted domain is connected, the composer archive should resolve at:

`https://listentomarcus.org/composer-archived/`
